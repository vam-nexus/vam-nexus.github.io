2/4 Week 3

🔄 Setup GitHub & Run Starter Code

🎯 Goal:

Set up version control and run the starter project code

❓ Why:

Version control is essential for tracking changes and collaborating with others, while the starter code provides a foundation for your project.

📝 Action Items:

  • Create a GitHub account at GitHub Signup (if you don't have one already)
  • Set up SSH for GitHub:
    1. Generate an SSH key: ssh-keygen -t ed25519 -C "your_email@example.com"
    2. Start the SSH agent: eval "$(ssh-agent -s)"
    3. Add your SSH key: ssh-add ~/.ssh/id_ed25519
    4. Copy your public key: cat ~/.ssh/id_ed25519.pub
    5. Add the key to your GitHub account: GitHub → Settings → SSH and GPG keys → New SSH key
  • Create your own GitHub repository:
    1. Go to https://github.com/new
    2. Name your repository (e.g., "my-ai-project" or something related to your idea)
    3. Set it to Public
    4. Click "Create repository"
  • Clone the starter project repository using SSH:
    1. Go to https://github.com/vam-nexus/starter-code
    2. Click the green "Code" button and copy the SSH URL
    3. In your terminal: git clone git@github.com:vam-nexus/starter-code.git
  • Copy the starter code to your repository:
    1. Copy the files from the starter code to your project folder
    2. Initialize your repository: git init
    3. Add your files: git add .
    4. Commit your files: git commit -m "Initial commit with starter code"
    5. Add your remote: git remote add origin git@github.com:your-username/your-repo-name.git
    6. Push to GitHub: git push -u origin main (or git push -u origin master depending on your default branch)
  • Navigate to the project folder and run the main script: cd your-repo-name python main.py
  • Take a screenshot of your github repository and upload it to Discord channel #📊｜week-3-outputs
  • Mark "Yes" in the Week 3 spreadsheet
